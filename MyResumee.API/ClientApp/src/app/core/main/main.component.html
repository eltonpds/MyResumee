Técnicas de Desenvolvimento


SPA

As aplicações de páginas simples (do inglês: Single Page Application) tendem a ser mais leves e rápidos pois as páginas ou componentes são carregados apenas quando o usuário as solicita onde o restante ta página permanece estático, economizando no consumo de banda do servidor.

Responsividade

Um site responsivo consiste em uma página que pode se adaptar aos mais variados tamanhos de telas diferentes, tanto para telas com resolução 1080p quanto 240p.

SEO

É um conjunto de estratégias elaboradas para potencializar e melhorar o posicionamento de um site nos resultados de busca fazendo com que o seu site fique mais próximo do topo.
